****************************************README***************************************


5 code files

1. "wavecluster1.ipynb" : implementation of wavecluster algorithm

2. "BaseLine.ipynb" : implementation of baseline algorithm

3. "PrivQT.ipynb" : implementation of private quantization algorithm

4. "PrivTHR.ipynb" : implementation of private threshold algorithm

5. "PrivTHRem.ipynb" : implementation of private threshold(exponential machanism) algorithm



Open each file and simply run the code to see the calculations and results.



"blobs_dataset.pkl" : contains dataset


"label_sets.pkl" and "number.txt" files store the results of normal wavecluster algorithm which is used in other algorithms for comparison.
